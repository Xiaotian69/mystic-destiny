一、项目概述

1.1 项目背景

随着移动互联网的快速发展，用户对于个性化、娱乐化内容的需求日益增长。命理、占星、塔罗等传统玄学文化在年轻一代中重新焕发活力，成为社交话题与精神寄托的重要载体。微信小程序以其无需下载、触达便捷、社交裂变能力强的特点，是承载此类文化产品的理想平台。

本项目旨在开发一款融合「周易算卦」「塔罗牌」「星座占星」三大核心功能于一体的微信小程序，用户只需输入生辰（出生年月日时），即可一键获取个性化命理解读、塔罗牌阵分析及星座运势，同时提供每日运势与行动建议。

1.2 项目目标

核心功能：用户输入生辰八字，自动生成算卦结果、塔罗牌阵、星座解析三合一报告

每日运势：基于用户生辰，每日自动推送今日运势与个性化建议

用户体验：界面神秘优雅，操作流程极简（最多3步），降低使用门槛

商业价值：为后期广告投放、会员付费、运势详解付费打下基础

3.2 周易算卦模块（可代码化逻辑）

3.2.1 八字计算算法

八字（四柱）由年柱、月柱、日柱、时柱各对应一组天干地支构成，每柱由「天干」+「地支」组成，共8个字，全部可通过数学公式精确计算：

年柱：(出生年 - 4) % 60 → 查六十甲子表，得天干地支

月柱：依据节气分界（用二十四节气精确月份），查天干推演表

日柱：以公历日期计算儒略日数，对60取余，查六十甲子表

时柱：将出生时间转换为十二地支时辰（子时=23:00-01:00，丑时=01:00-03:00…）

3.2.2 本命卦推算

数字命理法（先天八卦数）：

男命：（100 - 出生年后两位）/ 9，余数对应先天卦位

女命：（出生年后两位 + 5）/ 9，余数对应先天卦位

六十四卦查询：依据日柱+时柱天干地支组合，映射至六十四卦索引表

3.2.3 五行强弱分析

统计四柱8个字中金木水火土各属性出现次数，结合月令旺衰系数（固定数据表），计算五行力量百分比，用于生成"五行缺失"与"喜用神"建议。

3.3 塔罗牌模块（可代码化逻辑）

塔罗牌分析采用确定性生成（基于生辰的哈希算法），保证同一用户每次看到结果一致，增强可信度：

3.3.1 牌阵生成算法

步骤1：将出生日期字符串（如 "19950815"）转换为数字哈希值

步骤2：用哈希值对78（大小阿尔卡纳总牌数）取余，得到3张牌的索引（过去/现在/未来）

步骤3：判断正逆位：哈希值的对应位奇偶性决定正/逆位

步骤4：查塔罗牌数据库（JSON格式存储78张牌的关键词、牌义、正逆位释义）

3.3.2 数据结构示例

每张塔罗牌以JSON对象存储：{ id: 0, name: "愚者", keywords: ["开始","自由","冒险"], upright: "勇于迈出第一步...", reversed: "鲁莽、缺乏计划..." }

3.4 星座模块（可代码化逻辑）

太阳星座：依据出生月日查日期区间表（白羊座3.21-4.19，金牛座4.20-5.20…）

上升星座：依据出生时间+出生地经度（可用城市选择器替代），查黄道宫升星表

月亮星座：依据出生日期推算月球黄道位置（查预计算的月亮星历表）

运势内容：12星座 × 365天的运势文本，按星座+日期组合预生成并存储于云数据库

3.5 今日运势与建议

每日运势采用「模板引擎 + 动态变量」方案，由系统自动合成个性化文本：

基础模板：分事业/感情/财运/健康四个维度，各维度有10-15套文本模板

动态变量：依据当日八字（今日日柱时柱）与用户本命信息的五行关系，选择对应吉凶模板

运势评分：1-5星，基于当日与本命五行的相生相克关系自动计算

行动建议：如「今日宜：签约、出行；忌：动土、探病」，依据神煞数据表自动匹配

七、UI/UX 设计规范

7.1 视觉风格

整体设计语言采用「东方神秘美学」风格，融合传统水墨与现代扁平化设计：

主色调：深靛蓝 (#1A1033) + 星空紫 (#4B0082) 营造神秘夜空感

点缀色：金色 (#C8A951) 用于重要信息标注，象征灵性与智慧

背景：星空粒子动效背景（Canvas轻量实现），增强沉浸感

字体：中文使用「宋体/楷体」标题 + 系统默认正文，英文辅助信息使用衬线字体

图标：线条风格图标，六十四卦符号、星座星图使用SVG矢量图

7.2 交互设计原则

极简输入：一次性输入生辰，此后所有功能自动运行，不需要重复操作

即时反馈：提交生辰后1.5秒内完成计算并呈现结果，加载期间显示星象动画

渐进式披露：首屏展示摘要卡片，点击进入模块查看详细解读，避免信息过载

分享能力：每个解读结果可生成分享海报（Canvas绘制），引导社交传播

7.3 响应式适配

使用 rpx 单位确保 iPhone SE（小屏）到 iPhone Pro Max（大屏）自适应

关键布局断点：375rpx 基准，750rpx 满宽设计稿

Swiper 滑动翻页控制高度，避免内容溢出
